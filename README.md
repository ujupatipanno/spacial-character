# 기호 입력 (Special Character Input)

자주 쓰는 특수기호를 빠르게 입력할 수 있는 Obsidian 플러그인입니다.

## ✨ 주요 기능

- **트리거 기반 자동완성**: `/` (또는 사용자 정의 문자)를 입력하면 기호 목록이 표시됩니다
- **퍼지 검색**: 정확한 ID를 입력하지 않아도 기호를 찾을 수 있습니다 (예: `arr` → `arrow`)
- **하이라이트 표시**: 검색어와 매칭되는 문자를 강조하여 표시합니다
- **간단한 구조**: ID와 기호만으로 구성된 심플한 관리 시스템
- **순서 조정**: 기호의 순서를 자유롭게 변경할 수 있습니다

## 📦 설치 방법

### 수동 설치

1. 이 저장소를 다운로드합니다
2. `main.js`와 `manifest.json` 파일을 Obsidian vault의 `.obsidian/plugins/spacial-character/` 폴더에 복사합니다
3. Obsidian 설정에서 "커뮤니티 플러그인"으로 이동하여 플러그인을 활성화합니다

## 🚀 사용 방법

### 1. 기호 추가

1. 설정 탭으로 이동합니다 (설정 > 커뮤니티 플러그인 > 기호 입력)
2. "새 기호 추가" 섹션에서 **+ 새 기호** 버튼을 클릭합니다
3. **ID**: 검색용 식별자를 입력합니다 (예: `arrow-right`, `check`)
4. **기호**: 실제 삽입할 기호를 입력합니다 (예: `→`, `✓`)

### 2. 기호 입력

1. 에디터에서 트리거 문자를 입력합니다 (기본값: `/`)
2. 검색어를 입력합니다 (예: `arr`)
3. 자동완성 목록에서 원하는 기호를 선택합니다
4. 기호가 자동으로 삽입됩니다

### 3. 기호 관리

- **순서 변경**: ID 옆의 ↑ ↓ 버튼으로 기호 순서를 조정합니다
- **삭제**: 🗑️ 버튼으로 기호를 삭제합니다
- **수정**: ID나 기호를 직접 수정하면 자동으로 저장됩니다

## ⚙️ 설정

### 기본 설정

- **트리거 문자**: 자동완성을 시작할 문자 (기본값: `/`)
- **퍼지 검색 활성화**: 부분 매칭 검색 사용 여부 (기본값: 활성화)
- **매칭 하이라이트**: 검색어 매칭 부분 강조 표시 (기본값: 활성화)
